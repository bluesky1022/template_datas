🚀 QUICK START (3 Steps):

1️⃣  SETUP MONGODB:
   
   Option A - MongoDB Atlas (Recommended):
   • Visit: https://www.mongodb.com/cloud/atlas/register
   • Create free M0 cluster (512 MB)
   • Get connection string from "Connect" button
   
   Option B - Local MongoDB:
   • Download: https://www.mongodb.com/try/download/community
   • Or Docker: docker run -d -p 27017:27017 mongo

2️⃣  UPDATE .ENV FILE:
   
   Open .env and add:
   
   # For MongoDB Atlas:
   DATABASE_URL="mongodb+srv://user:pass@cluster.mongodb.net/elearning?retryWrites=true&w=majority"
   
   # For Local MongoDB:
   DATABASE_URL="mongodb://localhost:27017/elearning"

3️⃣  INITIALIZE DATABASE:
   
   npm install                    # If not already done
   npx prisma generate           # Generate MongoDB client
   npx prisma db push           # Create collections
   npm run dev                  # Start development server

🆘 TROUBLESHOOTING:

❌ "Cannot connect to database"
   → Check DATABASE_URL in .env file

❌ "Authentication failed"
   → Verify username/password in connection string

❌ "Could not connect to server"
   → Local: Is MongoDB running?
   → Atlas: Is your IP whitelisted?

❌ "npx command not recognized" (Windows)
   → Run PowerShell as Administrator:
   → Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
